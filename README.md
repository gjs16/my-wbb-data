Automated WeHoop Data RepositoryThis repository automatically fetches and updates Women's Basketball play-by-play data daily using GitHub Actions and the wehoop R package.Data SourcesThe data is stored in the data/ directory in Compressed CSV format (.csv.gz). This format is required to fit the massive NCAA datasets within GitHub's file size limits.LeagueSeasonFile PathWNBA2025data/wnba_pbp_2025.csv.gzNCAA WBB2024-25data/ncaa_wbb_pbp_2025.csv.gzNCAA WBB2025-26data/ncaa_wbb_pbp_2026.csv.gz(Note: In wehoop, the "season" year refers to the year the championship is played. So 2026 is the 2025-26 season.)How to use with ChatGPT / ClaudeTo analyze this data in an LLM, you need the Raw Link.Navigate to the data/ folder in this repository.Click on the file you want (e.g., ncaa_wbb_pbp_2026.csv.gz).Look for the "Download" button or the "Raw" button (sometimes inside a "..." menu on mobile).Right-click that button and select "Copy Link Address".Paste that link into ChatGPT/Claude with the prompt below.Example Prompt for ChatGPT"I am providing a link to a compressed CSV file containing NCAA Women's Basketball play-by-play data. Please load this dataset directly (pandas can read .csv.gz files automatically) and perform the following analysis..."[PASTE YOUR RAW LINK HERE]Setup InstructionsFork this repository to your own GitHub account.Go to the Actions tab.Click "I understand my workflows, go ahead and enable them".Select the Daily Data Refresh workflow on the left.Click Run workflow to trigger the first data pull immediately.
